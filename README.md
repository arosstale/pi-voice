# @artale/pi-voice

Voice input for pi. Record audio, transcribe with Whisper (Groq/OpenAI).

## Install
```bash
npm install -g @artale/pi-voice
```

## Tools
- **voice_capture** — Record and transcribe audio

## Commands
- `/voice` — Voice input

## Requirements
- SoX (`sox` command)
- GROQ_API_KEY or OPENAI_API_KEY
